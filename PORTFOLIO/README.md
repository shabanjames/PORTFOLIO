## Project Description
This project is a personal portfolio website created as part of the Web Development Lab.  
It demonstrates the use of **semantic HTML**, **CSS Flexbox**, **CSS Grid**, and **responsive design** principles.  
The site includes sections for About, Projects, and Contact, styled with CSS variables and media queries.

## Technologies Used
- HTML5  
- CSS3  
- Git & GitHub  
- GitHub Pages for deployment
